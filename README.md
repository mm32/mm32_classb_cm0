# mm32_classb_cm0

基于灵动 Cortex M0 内核 MCU 的 IEC60730 Class B 软件仓库。

## 支持MCU

- MM32F0020
    - Mini-F0020 (MM32F0020B1T)
- MM32SPIN0230
    - Motor-DK SPIN0230 (MM32SPIN0230B3TV)
- MM32SPIN0280
    - Mini-SPIN0280(MM32SPIN0280D6P)


## 支持工具链

- KEIL-MDK V5.23 (AC5 -O0)

- IAR-EWARM V8.22.1 (ICC -None)
